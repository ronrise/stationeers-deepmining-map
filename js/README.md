# Javascript Frontend

This directory contains the source code for the Javascript frontend of the project. 

building
## Prerequisites

Before building the project, ensure you have the following installed:
- Node.js (version 24 or higher)
- npm (Node Package Manager)

### nvm
If you are using `nvm` (Node Version Manager), you can install and use the required Node.js version with the following commands:
```bash
nvm install 24
nvm use 24
```

## Installation
To install the necessary dependencies, navigate to the `js` directory and run:
```bash
npm install
```

## Building the Project
To build the project, use the following command:
```bash
npm run build
```

This will create a `dist` directory containing the built files.

## Running the Development Server
To run a development server with live reloading, use:
```bash
npm run dev
```
